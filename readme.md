# HKSCS轉換Big5

HKSCS轉換Big5線上版 - [https://mollykannn.github.io/translate-hkscs-big5](https://mollykannn.github.io/translate-hkscs-big5)

字詞出自:
- [異體字/異形詞轉換](https://sntcm.wordpress.com/2009/12/15/%E5%B0%8F%E7%8B%BC%E6%A8%99%E6%BA%96/) By 李小狼

## 安裝 (Install)

```shell
yarn install
```

## 使用方法 (Usage)

建立檔案 (Create files)
```shell
yarn run build
```

運行 (Run)
```shell
yarn run dev
```
